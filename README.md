## How to open (no Node / npm needed)

1. Unzip `rajuhero.zip`
2. Open the `footbaaal` folder
3. Double-click **`index.html`**

### Optional localhost
- Windows: `START-WINDOWS.bat` → http://localhost:3000
- Mac: `START-MAC.command` → http://localhost:3000
